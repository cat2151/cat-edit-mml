# cat-edit-mml デモガイド

## クイックスタート

```bash
cargo run
```

## 期待される動作

エディタを実行すると、以下が表示されます：
- フルスクリーンのターミナルインターフェース
- 「MML Editor - Press ESC to exit」というタイトルの枠線付きウィンドウ
- 入力可能なテキストカーソル

## 機能デモンストレーション

### 1. 複数行テキスト入力
- 任意のテキストを入力
- Enterキーで新しい行を作成
- カーソルは入力に追従

### 2. カーソル移動
- **矢印キー** (←↑↓→): 任意の方向にカーソルを移動
- **Home**: 現在行の先頭へジャンプ
- **End**: 現在行の末尾へジャンプ
- **Page Up**: 1ページ上にスクロール
- **Page Down**: 1ページ下にスクロール

### 3. テキスト編集
- **Backspace**: カーソル前の文字を削除
- **Delete**: カーソル位置の文字を削除
- **Ctrl+H**: バックスペースの代替
- **任意の文字**: カーソル位置に挿入

### 4. 終了
- **ESC**: エディタを終了してシェルに戻る

## 試せるMMLコンテンツの例

オーディオ再生が実装されたら（AUDIO_PLAYBACK_PLAN.mdを参照）、次のようなMMLを編集できます：

```
C D E F G A B
```

または

```
T120 L4 O4
C E G > C < G E C
```

## 技術詳細

このエディタは以下を使用して構築されています：
- **ratatui 0.29**: TUIフレームワーク用
- **tui-textarea 0.7**: 複数行テキスト編集用
- **crossterm 0.28**: クロスプラットフォーム端末処理用

実装は`src/main.rs`にあり、約60行のコードです。

## パフォーマンス

エディタは軽量で応答性が高い：
- バイナリサイズ: 約1.1 MB（リリースビルド）
- メモリ使用量: 5 MB未満
- CPU使用率: 最小限（イベント駆動）
