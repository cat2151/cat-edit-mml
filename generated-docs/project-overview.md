# Project Overview

## プロジェクト概要
- Rustで構築された、MML（Music Macro Language）用のシンプルなTUIエディタです。
- テキスト編集、カーソル移動、MMLノート入力時の自動再生、MMLテンプレート切り替えなどの機能を提供します。
- コマンドラインから手軽にMMLの作曲と再生を試せる、ミニマムな体験を目指しています。

## 技術スタック
- フロントエンド:
    - **ratatui**: TUI (Text User Interface) を構築するためのフレームワークです。
    - **tui-textarea**: ratatui上で複数行のテキスト入力エリアを提供するウィジェットです。
    - **crossterm**: 端末操作（キー入力、カーソル移動、画面クリアなど）をクロスプラットフォームで実現するためのライブラリです。
- 音楽・オーディオ:
    - **cat-play-mml**: MMLパーサーとオーディオプレイヤー。本プロジェクトでは子プロセスとして呼び出し、MMLの自動再生に使用されます。（Windows専用、Nuked-OPMエンジン利用）
- 開発ツール:
    - **Cargo**: Rustプロジェクトのビルド、依存関係管理、テスト実行などを扱う標準ツールです。
- 言語機能:
    - **Rust 1.70以降**: 高い安全性、パフォーマンス、並行性を特徴とするプログラミング言語です。
- ビルドツール:
    - **Cargo**: Rustプロジェクトのビルドプロセスを管理します。
- テスト:
    - **Cargo (標準テスト機能)**: Rustの標準機能である`cargo test`コマンドを使用し、ユニットテストや統合テストを実行します。
